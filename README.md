# E-Commerce Sales Dashboard

## Overview
Developed an interactive Power BI dashboard using Excel and SQL data to analyze e-commerce sales performance, customer behavior, profitability, and regional trends.

## Tools & Technologies
- Power BI
- SQL
- Microsoft Excel

## Key Metrics
- Total Sales: 2.15M
- Total Orders: 2K
- Total Customers: 200
- Average Discount: 11.85%

## Features
- Sales analysis by category and customer segment
- Profit analysis by region and category
- Payment mode distribution tracking
- Interactive filters for category, region, and month
- Revenue trend monitoring
